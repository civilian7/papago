# papago
Naver papago 번역 서비스 클라이언트

API 키는 NAVER에서 발급받은 후 papago.ini 파일에 기록한다.
API는 https://developers.naver.com/docs/papago/papago-nmt-overview.md 에서 발급 받을 수 있다
(네이버 아이디 필수로 필요함)

[KEY]
ID= (발급받은 Client ID)
SECRET= (발급받은 Client Secret Key)

[OPTION]
SOURCE=0
TARGET=1
AUTOCOPY=1

[POSITION]
LEFT=342
TOP=165
